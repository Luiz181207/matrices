# Matrices

Exercises focused on matrix manipulation and two-dimensional data structures in Python.

Topics covered:

* Matrix creation
* User input matrices
* Random matrix generation
* Matrix traversal
* Row and column operations
* Matrix calculations

## Concepts Practiced

* Nested loops
* Two-dimensional arrays
* Matrix indexing
* Data organization
* Mathematical operations

## Goal

Develop the ability to work with two-dimensional data structures and strengthen logical reasoning through matrix-based problems.
